## 静态引入

推荐使用快速上手中的方式，进行新项目开发。静态引入的方式比较适合老的应用。

### 下载BUI

我们提供了cdn下载地址(右击-链接另存为就可以下载,也可以直接用cdn地址)

- js: http://b0.hucdn.com/bui/vue-2.1.10.js
- min.js: http://b0.hucdn.com/bui/vue-2.1.10.min.js
